About This Project

This project addresses critical challenges like water waste and food security by integrating environmental sensors with IoT technology. It uses temperature, humidity, soil moisture, and water level sensors connected to an ESP32 microcontroller that streams real-time data to a backend and user interfaces.

How It Works

Sensors → ESP32 → FastAPI → Website + Blynk App + WhatsApp Alerts

Learning Journey and Challenges

Building this project was a valuable learning experience involving hardware-software integration, real-time data handling, API communication, and web development. Challenges included managing sensor accuracy, ensuring stable Wi-Fi connectivity, and creating a user-friendly interface for monitoring and alerts.
